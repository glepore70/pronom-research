What is a PSSG file?
Graphics file used by games created with the EGO Engine, a race car game engine developed by Codemasters; contains textures for game objects, which may be stored in .DDS, .TGA, or .BMP files; used for racing car skins, track graphics, and other game graphics.

More Information
The EGO Engine was used to develop games such as:

Colin McRae: DiRT
Colin McRae: DiRT 2
Colin McRae: DiRT 3
Race Driver: GRID
F1 2010
Operation Flashpoint: Dragon Rising
Operation Flashpoint: Red River
The textures in PSSG files can be edited using several editors, including the PSSG Editor and the F1 2010 Textures Editor.
