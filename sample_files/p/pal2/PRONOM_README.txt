Turbo Packer (or maybe there's only Turbo Packer Plus?) is a file compression utility for Atari ST computers. It was developed by Wolfgang Mayerle, and published by Data Becker. Most of the information about it is in German.

It apparently supports transparent file decompression.

Identification
According to the Ancient software, files begin with ASCII "TPWM".
