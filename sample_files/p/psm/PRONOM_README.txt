Epic Megagames MASI - also known as ProTracker Studio Module although, confusingly, ProTracker was probably not used to create .psm files - is an audio file format used in a number of games created by Epic (Jazz Jackrabbit, Epic Pinball, Extreme Pinball, Silverball, Sinaria and One Must Fall).
