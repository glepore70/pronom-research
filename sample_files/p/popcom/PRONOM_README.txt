LHA is a freeware compression utility and associated file format. It was created in 1988 by nihongo|Haruyasu Yoshizaki|吉崎栄泰|Yoshizaki Haruyasu, and originally named LHarc.


PMarc extensions[edit]
These compression methods are created by PMarc, a CP/M archiver created by Miyo. The archive usually has a .PMA extension.

pc1
PopCom compressed executable archive. Details unknown.
