PageStream (originally Publishing Partner) is a desktop publishing software package by Grasshopper LLC (United States), currently available for a variety of operating systems, including Windows, Linux, and Macintosh. The software was originally released under the name Publishing Partner for the Atari ST in 1986.[1][2][3] It was also used on the Amiga platform starting in 1989.[4] According to the official website, the latest version is PageStream 5.0.5.8, released October 11, 2010. It is currently available only for Windows, Linux, and Macintosh OSX.

Autotracing application BME included.
