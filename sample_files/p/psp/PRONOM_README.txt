ProShape (also written PRO-SHAPE) is a vector graphics editor for Windows 2.x/3.x. It was developed by John Hazard and Hazard Software. It has a native .psp file format.

Identification
ProShape v1.0 PSP files start with ASCII "ProShape V1.0 Drawing", then bytes 0x0d 0x0a 0x1a.

ProShape v1.1 PSP files start with ASCII "ProShape V1.x Drawing", then bytes 0x0d 0x0a 0x1a.
