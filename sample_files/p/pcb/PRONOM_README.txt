What is CIRCAD?
Software for making printed circuit boards. With CIRCAD
you can do schematic capture and layout in a graphical
user interface under DOS. CIRCAD also has vector based
drawing capability.


Is CIRCAD shareware?
The demo is freeware. The professional version is $995.
Students and Hobbyists can get a $299 version.


Where do I get CIRCAD?
Use ftp on mobius.gmu.edu in /pub/circad for the demo version.
