ReportSmith is a database reporting and query tool that is the perfect companion to your PC data management software. Its WYSIWYG "live" data approach to reporting provides users with a drag and drop, visual interface that makes report creation and customization a snap.

ReportSmith was originally written by ReportSmith Inc, which was then bought by Borland. Version 2.0 was the first retail release.
