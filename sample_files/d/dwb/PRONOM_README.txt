File Information
VariCAD's native proprietary binary format is in the .DWB format. DWB files have the "870100" beginning at byte 2. Byte one seems to indicate version. Version 7 header is "32870100" Version 9 header is "34870100", the current version of VariCAD 2020, files begin with "4C870100"
