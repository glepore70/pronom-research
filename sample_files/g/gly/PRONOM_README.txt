Glossary file used by Microsoft Word, a word processing program; includes data that helps convert glossary terms between versions of Word.

The files 50convrt.gly, 55convrt.gly, and 60convrt.gly are provided with the MS Office for Windows 95 Resource Kit. They help convert Word 5.0, 5.5, and 6.0 for MS-DOS glossary entries into AutoText entries in Word 7.0.
