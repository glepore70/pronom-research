.gtk files are modules produced by Graoumf Tracker for the Atari Falcon 030.

http://fileformats.archiveteam.org/wiki/Graoumf_Tracker_module


.gt2 files are modules produced by Graoumf Tracker 2, the "sequel" to Graoumf Tracker for the Atari Falcon 030. Created by the same author, Laurent de Soras, Graoumf Tracker 2 runs on Microsoft Windows and was open-sourced in 2002 due to the author's lack of time to work on the project.

http://fileformats.archiveteam.org/wiki/Graoumf_Tracker_2_module
