What is inx file? How to open inx files?
File type specification:

Various data file type icon Various data file type

The inx file extension is associated with the InstallShield utility that allows users to create application installation packages for Microsoft Windows operating system.

The .inx file is present in installation packages created by InstallShield software and stores compiled installation rules. The .rul file represents the un-compiled version of this file.
