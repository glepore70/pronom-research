ICDRAW icon is an icon format associated with the ICDRAW icon editor for the Atari Falcon computer.

There are two kinds: "single icon files" (.IBI), and "group icon files" (.IB3).

Identification
IBI files begin with ASCII "ICBI", and IB3 files begin with "ICB3".
