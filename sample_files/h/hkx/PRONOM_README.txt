


=== HKX File format ===

The HKX file format (Havok Packed File) is defined by its header.

The header contains :
 - 64 bytes of pure header data
 - three sections
	classname
	types
	data

=== HKX pure header ==
Constant value		57E0 E057
				10C0 C010
