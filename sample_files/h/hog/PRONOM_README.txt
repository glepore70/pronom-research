This is the specifications for the archive file which is handled by the Game Extractor plugin Archive_HOG_DHF.

Endian Order:	Little Endian
Compression:	None
Encryption:	None


This format is used by the following Games:
Descent	*.hog
Descent 2	*.hog


This is the format specification information:
 	
3 - Header (DHF)

// for each file
  13 - Filename (null)
  4 - Size
  X - File Data
