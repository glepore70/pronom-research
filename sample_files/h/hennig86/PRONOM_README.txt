Hennig86 Format:

A simple Hennig86 file looks like this:

xread
'an optional text string in single quote chars'
10 5
TaxonA   0000000000
TaxonB   0010111000
TaxonC   1011110000
TaxonD   1111111000
TaxonE   1111111000

;

XREAD and  NSTATES commands are case insensitive.
