WinSysEx v3.0
WinSysEx v3.0 - A shareware, programmable MIDI message editor for MS-Windows 3.1. Create your own synth patch editor! Features definable spin controls, scrollbars, checkboxes and comboboxes; automatic checksum calculations; equations; variables.

Examples are included for Roland SCC-1, Yamaha DX-100, Korg DW-8000, EMU Proteus / Wave Blaster. $39.00.
