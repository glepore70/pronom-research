BAMG
Bidimensional Anisotropic Mesh Generator
BAMG is a data directory of files that can be used with BAMG, a program for generating 2D meshes that can be used to define the geometry for the the finite element package FREEFEM++

https://people.math.sc.edu/Burkardt/data/bamg/bamg.html




Mesh file data structure
The mesh data structure, output of a mesh generation algorithm, refers to the geometric data structure and in some case to another mesh data structure.
