What is scel file? How to open scel files?
File type specification:

Various data file type icon Various data file type

The scel file extension is associated with the Sogou Pinyin input method editor developed by Sogou a Chinese company and same named search engine.

The .scel file stores cell dictionary that adds new words to editor.
