Description
Reality Adlib Tracker is a 9-channel OPL2 adlib tracker for MS-DOS. It produces modules with the extension .rad, which can contain up to 31 instruments and up to 32 patterns. Version 2 supports OPL3 and supports more instruments and patterns.
