
DeskMate (Song)
DeskMate was a software application that provided an operating environment that competed with early versions of Microsoft Windows. It originally was made for Tandy's TRS-DOS Operating System and for their TRS-80 line of computers, but eventually shifted to PC. The identifying characters used for this association are - Hex: 0C 53 4E 47, ASCII: SNG Related links: Wikipedia DeskMate Page
