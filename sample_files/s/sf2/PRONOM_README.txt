File Format
Name	SoundFont 2.0
Ontology	
Electronic File Formats
Audio and Music
SoundFont 2.0
Extension(s)	.sf2
SoundFont 2.0 is a format used to store instrument patches for MIDI synthesisers. The SoundFont 2.0 format is based on RIFF; the RIFF type identifier is "sfbk".
