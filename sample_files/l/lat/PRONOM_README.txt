Description:	Micro-Lathe for Windows 3.1 is an easy-to-use 3D modeling program that allows you to create 3D objects using the metaphor of a Carpenter's Lathe. That is, you draw the outline in a flat, profile view, and Micro-Lathe will render your drawing as a solid object.
