The .opj file extension is normally associated with OrCAD PCB Designer. OPJ files normally contain printed circuit board projects that are used by electricians and mechanics to serve as a guide in installing or creating a wiring system for various electronic products and automobiles.

A project file (.OPJ) is a container for the design file (.DSN). In addition, a project file also includes simulation profile and layout information.
