Ontology	
Electronic File Formats
Compression
NPack
Released	1988
NPack is a file compression utility for DOS. It was developed by Stac Electronics and later acquired by Symantic in 1991.

Identification
Files begin with ASCII string MSTSM

An NPACK.EXE utility was included with some Symantec products, such as Norton Desktop 1.0 and 2.0 for Windows.
