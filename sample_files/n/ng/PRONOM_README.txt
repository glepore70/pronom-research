Release of the Norton Desktop software program prompted the development of the Norton Guide Online Documentation Database file type by Symantec Corporation. Internal website statistics show that NG files are most popular with users in United States and those running the Windows 10 operating system. A high percentage of these users browse the web using Google Chrome.

NG file is a Norton Guide. Norton Guides are an old hypertext engine developed by the Peter Norton Company.
